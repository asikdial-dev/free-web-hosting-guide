## 📋 How to Contribute

### 1. 🐛 Report Issues
- Found a broken link? [Open an issue](https://github.com/asikdial-tech/free-web-hosting-guide/issues/new)
- Service no longer free? Let us know!
- Information outdated? Help us update it!

### 2. ➕ Add a New Service

**Before adding, please verify:**
- [ ] The service has a **genuinely free tier** (not just a free trial)
- [ ] The service is **currently active and accepting new users**
- [ ] The service is **not already listed**
- [ ] You have **personally tested** or **verified** the service

**Required information:**


### 3. 📝 Steps to Contribute

1. **Fork** this repository
2. **Create** a new branch: `git checkout -b add-service-name`
3. **Add** the service in the correct category
4. **Commit** your changes: `git commit -m "Add [Service Name] to [Category]"`
5. **Push** to your fork: `git push origin add-service-name`
6. **Open** a Pull Request with a clear description

---

## 📏 Guidelines

### ✅ Do:
- Keep descriptions concise and factual
- Include accurate storage/bandwidth numbers
- Mention if ads are shown
- Note any important limitations
- Use proper Markdown formatting
- Place services in the correct category

### ❌ Don't:
- Add referral or affiliate links
- Add services requiring a credit card for the free tier
- Use exaggerated or misleading language
- Add duplicate services
- Submit services you haven't verified

---

## 🏷️ Categories

| Category | What Belongs Here |
|---|---|
| **Full-Stack Hosting** | PHP, MySQL, Python, Node.js hosting with server-side capabilities |
| **Static Site Hosting** | HTML/CSS/JS only, JAMstack, SSG-based hosting |
| **Cloud & PaaS** | Container-based, serverless, or platform-as-a-service hosting |
| **Website Builders** | No-code/low-code platforms with hosting included |
| **Documentation Hosting** | Specialized documentation/wiki hosting |
| **New Additions** | Recently launched or discovered services |

## 💜 Code of Conduct

Be respectful, inclusive, and constructive. We're all here to help each other.
